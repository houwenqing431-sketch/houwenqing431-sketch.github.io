# My Technical Blog 🚀

这是我的个人技术博客仓库，基于 **Hexo** 框架搭建，采用 **Fluid** 主题。

## 📝 博客简介
- **博主**: houwenqing431-sketch
- **主要内容**: Java 后端开发、算法笔记（如素数筛）、设计模式等学习记录。
- **访问地址**: [https://houwenqing431-sketch.github.io](https://houwenqing431-sketch.github.io)

## 🛠️ 本地开发环境
如果你需要在其他电脑上克隆并运行此博客，请确保已安装：
- Node.js
- Git
- Hexo-cli

### 常用命令
```bash
# 安装依赖
npm install

# 新建文章
hexo new "文章标题"

# 本地预览
hexo clean && hexo s

# 部署到 GitHub Pages
hexo clean && hexo g && hexo d
