Java学习笔记-素数筛

在近期的 Java 学习中，总结了一个非常实用的知识点：快速查找素数的埃氏筛算法。特此记录，方便日后复习回顾。

素数查找：埃拉托斯特尼筛法

课堂上展示了 2 到 120 的数字网格演示，本质就是经典的埃氏素数筛。

核心思想

从数字 2 开始，把每个素数的所有倍数全部标记为非素数。
不断剔除合数，最后没有被标记的数字，就是区间内所有素数。

Java 代码实现

public class PrimeSieve {
    public static void main(String[] args) {
        int max = 120;
        boolean[] isPrime = new boolean[max + 1];
        
        // 初始化：默认 2 以上全部为素数
        for (int i = 2; i <= max; i++) {
            isPrime[i] = true;
        }

        // 筛选非素数
        for (int p = 2; p * p <= max; p++) {
            if (isPrime[p]) {
                // 将当前素数的倍数标记为非素数
                for (int i = p * p; i <= max; i += p) {
                    isPrime[i] = false;
                }
            }
        }

        // 遍历输出所有素数
        System.out.println("2 到 120 的素数有：");
        for (int i = 2; i <= max; i++) {
            if (isPrime[i]) {
                System.out.print(i + " ");
            }
        }
    }
}
